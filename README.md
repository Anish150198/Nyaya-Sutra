# Nyaya-Sutra
Nyaya-Flow is an agentic AI ecosystem designed to bridge the justice gap in India. By leveraging the Databricks Data Intelligence Platform, it automates the transition from raw human testimony in local languages to court-ready legal documents (FIRs, Affidavits, Plaints) aligned with the new Bharatiya Nyaya Sanhita (BNS) 2023 framework.
